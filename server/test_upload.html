Test Upload to hepius.co

This will test if your server can receive image uploads.

Troubleshooting Steps:

  1. Check file location:
    upload.php should be at: public_html/scoliosis-uploads/upload.php
  2. Check permissions:
    • scoliosis-uploads folder: 755
    • images folder: 755
    • upload.php: 644
  3. Check .htaccess:
    Make sure .htaccess is uploaded to: public_html/scoliosis-uploads/.htaccess
  4. Check PHP version:
    Server should have PHP 7.0 or higher

What should happen:

  • You should see a success message above
  • A test image file should appear in: scoliosis-uploads/images/
  • A metadata JSON file should be created
  • An upload_log.txt file should be created/updated
